# claude-plugins

> Claude Code plugin marketplace by [@theabecaster](https://github.com/theabecaster)

## Install

Add this marketplace once inside Claude Code:

```
/plugin marketplace add theabecaster/claude-plugins
```

Then install any plugin:

```
/plugin install claude-learns@abe-plugins
```

## Plugins

### claude-learns
Claude Code that learns from you. Auto-captures your corrections as preferences in `CLAUDE.md`, audits for conflicts and expiry.

→ [github.com/theabecaster/claude-learns](https://github.com/theabecaster/claude-learns)
